대구광역시장 선거 관련 여론조사를 기록합니다.

여론조사심의위원회 공개 자료와 언론 보도를 기반으로 작성합니다. 수치 해석이 과장되지 않도록 조사 조건을 함께 기록합니다.

## 기록 형식

각 여론조사마다 다음 정보를 함께 기록합니다.

| 항목 | 설명 |
| --- | --- |
| 조사기관 | 실제 조사를 수행한 기관 |
| 의뢰기관 | 조사를 의뢰한 기관 또는 매체 |
| 조사일 | 실시 날짜 |
| 표본 크기 | 응답자 수 |
| 표본 방법 | 전화면접, ARS, 온라인 등 |
| 오차범위 | 95% 신뢰수준 기준 |
| 출처 | 원문 링크 또는 source id |
